Welcome to Galv's documentation!

The current documentation is for the Galv monorepo.
This has been split into separate repositories with the following structure:

* `Specification <https://github.com/Battery-Intelligence-Lab/galv-spec/>`_ - The specification for the Galv project
* `Backend <https://github.com/Battery-Intelligence-Lab/galv-backend/>`_ - The Django REST Framework API backend
* `Frontend <https://github.com/Battery-Intelligence-Lab/galv-frontend/>`_ - The React frontend
* `Harvester <https://github.com/Battery-Intelligence-Lab/galv-harvester/>`_ - The data harvester Python program
